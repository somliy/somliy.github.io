安装后配置

首先更换国内源
sudo pacman -Syy
sudo pacman-mirrors -i -c China -m rank  #选一个清华源就行
sudo pacman -Syyu


先安装vim
sudo pacman -S vim


安装ssh
systemctl enable sshd.service 开机启动
systemctl start sshd.service 立即启动
systemctl restart sshd.service 立即重启


添加arch源
sudo vim /etc/pacman.conf
把下边这个添加进去
[archlinuxcn]
SigLevel = Optional TrustedOnly
Server = https://mirrors.ustc.edu.cn/archlinuxcn/$arch
然后
sudo pacman -Syy && sudo pacman -S archlinuxcn-keyring


排列源
sudo pacman-mirrors -c China


安装zsh
sudo pacman -S git -y
sudo pacman -S zsh -y
sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
chsh -s /bin/zsha

安装archlinuxcn-keyring
sudo pacman -S archlinuxcn-keyring -y
如果安装失败,则执行以下命令:
 sudo pacman -Syu haveged
 sudo systemctl start haveged
 sudo systemctl enable haveged
 sudo rm -rf /etc/pacman.d/gnupg
 sudo pacman-key --init
 sudo pacman-key --populate archlinux
 sudo pacman -S archlinuxcn-keyring
 sudo pacman-key --populate archlinuxcn


安装yaourt以及pacaur
sudo pacman -S yaourt pacaur -y
导入GPG Key
sudo pacman -Syy && sudo pacman -S archlinuxcn-keyring

此后就可以去安装自己想要使用软件啦！！

安装docker
sudo pacman -S docker

启动
sudo systemctl start docker

查看状态
sudo systemctl status docker

使用阿里源加速
sudo mkdir -p /etc/docker
sudo tee /etc/docker/daemon.json <<-'EOF'
{
  "registry-mirrors"=["https://82qv1mpy.mirror.aliyuncs.com"]
}
EOF
sudo systemctl daemon-reload
sudo systemctl restart docker



创建docker用户组
sudo groupadd docker
应用用户加入docker用户组
sudo usermod -aG docker ${USER}
重启docker服务
sudo systemctl restart docker
切换或者退出当前账户再从新登入
su root
su somliy
查看组中的人员
cat  /etc/group | grep docker

sudo pacman -S fcitx-im # 全部安装
sudo pacman -S fcitx-configtool # 图形化配置工具

设置中文输入法环境变量，编辑~/.xprofile文件，增加下面几行(如果文件不存在，则新建)
export GTK_IM_MODULE=fcitx
export QT_IM_MODULE=fcitx
export XMODIFIERS="@im=fcitx"

即时聊天工具——wechat
Github上electronic-wechat开源项目(据说比腾讯官方开发的要好-) 在安装前需要安装node.js不然不能使用其中的命令安装 安装 Node.js 的最佳方式是使用 nvm。
cURL:
curl https://raw.github.com/creationix/nvm/master/install.sh | sh
Wget:
wget -qO- https://raw.github.com/creationix/nvm/master/install.sh | sh
安装完成后，重启终端并执行下列命令即可安装 Node.js。
nvm install stable
若出现zsh=command not found=nvm
export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && . "$NVM_DIR/nvm.sh" # This loads nvm
写入 ~/.zshrc
source ~/.zshrc

安装UFW
sudo pacman -S ufw

安装额外规则（建議安裝）：
sudo pacman -S ufw-extras

启用
sudo ufw enable
启用 ufw 防火墙
sudo systemctl enable ufw.service


关闭
sudo ufw disable 

查看防火墙状态
sudo ufw status

开启/禁用相应端口或服务举例
sudo ufw allow 80 允许外部访问80端口

sudo ufw delete allow 80 禁止外部访问80 端口

sudo ufw allow from 192.168.1.1 允许此IP访问所有的本机端口
